Content основной контент элемента
padding пространство между содержиымым и границей элемента
border - граница , которая окружает элемент
Margin - пространство между соседними элементами
**

Пример в CSS:

.box {

  width: 200px;

  padding: 20px;

  border: 5px solid black;

  margin: 10px;

}
